# Notes Application

Приложение для создания и управления заметками.

## Функционал
- Просмотр списка всех заметок.
- Создание новой заметки.
- Редактирование существующих заметок.
- Удаление заметок.
- Постоянное хранение данных (Room).
- **Сохранение черновика**: Если вы начали создавать заметку и вышли, не сохранив ее, при следующем создании данные восстановятся.
- **Поиск**: Поиск заметок по заголовку и содержимому.
- **Теги и фильтрация**: Возможность добавлять теги к заметкам и фильтровать список по ним.
- **Изображения**: Возможность прикрепить изображение к заметке.
- **Обновленная иконка**: Современная адаптивная иконка приложения в стиле заметок.
- **Уведомления**: Возможность установить напоминание на определенное время с поддержкой звукового и вибросигнала.
- **Удаление всех заметок**: Быстрое удаление всей базы данных заметок с подтверждением действия.

## Технологический стек
- **Kotlin** - основной язык разработки.
- **Jetpack Compose** - современный UI фреймворк.
- **Room** - библиотека для работы с локальной базой данных SQLite.
- **Navigation Compose** - навигация между экранами.
- **Coil** - загрузка и отображение изображений.
- **AlarmManager** - планирование уведомлений.
- **Hilt** - внедрение зависимостей (Dependency Injection).
- **MVVM** (Model-View-ViewModel) - архитектурный паттерн.
- **Clean Architecture** - разделение на слои (Data, Domain, Presentation).

## Аргументация выбора библиотек
- **Hilt**: Стандартное решение от Google для DI, упрощающее управление зависимостями и улучшающее тестируемость.
- **Room**: Стандартное решение для Android для работы с БД.
- **Jetpack Compose**: Ускоряет разработку UI и делает код более читаемым и поддерживаемым по сравнению с традиционными View.
- **Navigation Compose**: Нативное решение для навигации в Compose-приложениях.
- **ViewModel & Flow**: Обеспечивают реактивное обновление UI и сохранение состояния при поворотах экрана.

## Инструкция по сборке
1. Склонируйте репозиторий.
2. Откройте проект в **Android Studio** (рекомендуется версия Ladybug или новее).
3. Дождитесь завершения Gradle Sync.
4. Выберите модуль `app` и нажмите **Run**.
